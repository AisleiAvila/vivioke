# Mídias locais

Coloque nesta pasta os arquivos de mídia usados pelo player.

Arquivos esperados pelo seed padrão:

- `parabens.mp4`
- `asa-branca.mp4`
- `stand-by-me.mp4`

A aplicação acessa esses arquivos pela rota `/media/*`.
Exemplo: `media/parabens.mp4` -> `instrumentalUrl: /media/parabens.mp4`
